Cookie Consent Banner
==============================
This is Drupal 7 custom module and it used to customize the way or look the cookie banner and all options given by EU Cookie Compliance (GDPR Compliance).

How it works.
=============
The module provides a custom cookie storage option settings in "Store record of consent" and the customised template to change the banner design over page.

Installation.
=============
1. Install and enable https://www.drupal.org/project/eu_cookie_compliance module beacuse of dependencies.

2. Unzip the files to the "sites/all/modules/custom" directory and enable the module.

3. Go to the admin/config/system/eu-cookie-compliance page to configure and
   enable the banner
